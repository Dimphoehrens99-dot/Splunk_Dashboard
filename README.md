# Splunk Dashboard

## Objective

The objective of this project is to filter data in the manner of a Security Operations Center (SOC) analyst, utilizing Splunk. A splunk dashboard will be employed to summarize the data, presenting it in a relevant and easy-to-read format with Alerts configured.

### Skills Learned

- Spluk SIEM practical application of features.
- Splunk Dashboard configuration
- Splunk Alert Configuration
- SPL

### Tools Used

- Splunk (SIEM)
